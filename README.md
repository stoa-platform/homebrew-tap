# STOA Platform Homebrew Tap

Official Homebrew tap for STOA Platform tools.

## Installation

```bash
brew tap stoa-platform/tap
brew install stoactl
```

Or install directly:

```bash
brew install stoa-platform/tap/stoactl
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `stoactl` | GitOps-native CLI for STOA Platform |

## Updating

```bash
brew update
brew upgrade stoactl
```
